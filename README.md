# Snake Game
This is small snake game developed in cpp language. For input (keypad stokes) and output (terminal display) we are using ncurses library API's.
## Build & Run -
Clone the repository and run following commands - 
```
mkdir -p build
cd build
cmake ..
make
./src/snake_game
```
